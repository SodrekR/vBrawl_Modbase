# Get Started TLDR
1. Get this Brawl build by downloading this Github Repository.
2. Download [GCTRealMate.exe](https://github.com/Project-Plus-Development-Team/GCTRealMate) and put it into the `vBrawl_Modbase\SDCardRoot\codes\` folder

# Brawl-Modbase
This modpack is based off of [KJP's BrawlEX build](https://drive.google.com/file/d/1sNiVAk2UwAVTXFbjwx4ee0eCyUfp4Bfv/view) for vanilla Brawl, with the [ExConfigs](https://discord.com/channels/1325321386557833316/1346170446915305574/1455669585696002262) for the characters so we can use [BrawlInstaller](https://github.com/squidgy617/BrawlInstaller) to mod this base Modpack.

## GCTRealMate.exe
You need to download [GCTRealMate.exe](https://github.com/Project-Plus-Development-Team/GCTRealMate) to generate the codes and for BrawlInstaller to work within this modpack.

Put the downloaded GCTRealMate.exe within the `vBrawl_Modbase\SDCardRoot\codes\` folder.

I didn't included GCTRealMate.exe here because I don't want to host any executables 
as you should also only download executables from direct trusted sources.
