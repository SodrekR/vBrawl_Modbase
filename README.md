# Brawl-Modbase
This modpack is based off of [KJP's BrawlEX build](https://drive.google.com/file/d/1sNiVAk2UwAVTXFbjwx4ee0eCyUfp4Bfv/view) for vanilla Brawl.

## GCTRealMate
I removed GCTRealMate.exe because I don't want to host any executables 
as you should also only download executables from direct trusted sources.

You can download GCTRealMate [here](https://github.com/Project-Plus-Development-Team/GCTRealMate).

I recommend you to put it within the `SDCardRoot/codes/` folder where it also originaly was within KJP's BrawlEX build.
