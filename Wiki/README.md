# Wiki
Here you will find rules that we follow when modding this Modpack.

## Stages
Different kind of stage mods can be loaded when holding a specific button (L, R, Z) while confirming a Stage. 

At the moment we add all custom stages (randomly chosen) to the normal selection where no button combination is held.

The original Stage without mods can be selected when holding Z while confirming the stage.
